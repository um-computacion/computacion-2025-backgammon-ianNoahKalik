#  Prompts utilizados para documentación del proyecto

Este archivo registra los intercambios con herramientas de IA utilizados para redactar la documentación del sistema Backgammon. La IA fue utilizada como asistente técnico para generar archivos explicativos, estructurar contenidos y presentar el proyecto de forma profesional.

---

##  Prompt 1: Justificación técnica del sistema

- **Herramienta usada**: Microsoft Copilot  
- **Prompt exacto**:  
  > Quiero que este muy completo.

- **Instrucciones del sistema**:  
  Solicité que el archivo `JUSTIFICACION.md` incluya diseño general, arquitectura, clases, atributos, excepciones, testing, principios SOLID y evolución del sistema.

- **Respuesta de la IA**:  
  Se generó el archivo `JUSTIFICACION.md` con:
  - Descripción modular del sistema  
  - Justificación de clases y atributos  
  - Manejo de errores con excepciones específicas  
  - Estrategias de testing y cobertura  
  - Aplicación de principios SOLID  
  - Diagrama UML y evolución del diseño

- **Uso de la salida**:  
   Usada con modificaciones menores

- **Archivo afectado**:  
  `JUSTIFICACION.md`

##  Prompt 2: Solicitud de redacción completa del README

- **Herramienta usada**: Microsoft Copilot  
- **Prompt exacto**:  
  > Me gustaría que me redactes un `README.md` completo y profesional para mi proyecto Backgammon. Tiene que incluir descripción, reglas implementadas, estructura del proyecto, instrucciones de instalación y ejecución (CLI y Pygame), testing, cobertura, desarrollo y contribuciones.

- **Instrucciones del sistema**:  
  Solicité que el archivo sea claro, técnico y listo para entrega final.

- **Respuesta de la IA**:  
  Se generó el archivo `README.md` con:
  - Descripción general del proyecto y reglas del juego  
  - Estructura detallada del repositorio  
  - Instrucciones de instalación y ejecución para ambos modos  
  - Sección de testing con cobertura y comandos  
  - Linting, documentación, contribuciones y licencia

- **Uso de la salida**:  
  Usada sin cambios

- **Archivo afectado**:  
  `README.md`
